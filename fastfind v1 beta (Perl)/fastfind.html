NAME
	fastfind - finds untaken three letter domains


SYNOPSIS
	fastfind [options] [file]

DESCRIPTION
	Finds all three letter domains that are still unregistered
	by digging up information on them from registrars.  The
	output of the program goes in available.txt by default if
	an alternate file isn't stated when the program is invoked.
	

	-f, --finish [domain]
		ends the program at the domain given.

	-h, --help
		displays the help dialog and then exits.

	-e, --extensions [extension ...]
		specifies which domain extensions to search for.

	-s, --start [domain]
		starts the program at the domain given.
	
	-t, --timeless
		report the output without the timestamps.

	-v, --verbose
		display more verbal output throughout the program.

	-V, --version
		display version information and then exits.
